Zadania do wykonania

1. Dodaj menu kontekstowe do pliku main.xml. Pamiętaj o indywidualnej nazwie
identyfikatorach atrybutów.

3. Stwórz obsługę menu kontekstowego w pliku MainActivity.java. Pamiętaj o
indywidualnym nazewnictwie zmiennych.

5. Narysuj szachownice zgodnie ze wzorem narysowanej klasy. Szachownica
poprawnie będzie wyświetlana w lokalizacji pionowej urządzenia. Twoim zadaniem
jest znaleźć rozwiązanie aby szachownica po zmianie orientacji ekranu nadal
wyglądała tak samo jak przy orientacji pionowej. W kodzie aplikacji została pobrana
tylko szerokość ekranu telefonu.
